# 平台运维文档索引

> 文档类型：总览文档
> 适用范围：平台状态、维护开关、首页公告、管理员公告配置与运行期运维入口
> 当前状态：现行专题导航

本文档用于登记平台运维相关的长期事实入口。这里的“平台运维”指面向管理员和玩家的站点状态、维护通知、公告展示与维护期间对局限制，不包含具体卡效、房间监控明细或生产发布 checklist。

## 当前文档

| 文档                                                         | 类型           | 维护边界                                                                       |
| ------------------------------------------------------------ | -------------- | ------------------------------------------------------------------------------ |
| [平台状态、维护开关与公告](site-status-and-announcements.md) | 需求与设计现状 | 维护开关、公开站点状态、公告类型、首页展示、管理员配置入口和维护期间新对局限制 |

## 维护规则

- 管理员配置类能力优先记录在本专题；只有联机房间运行状态、观战、回放或同步协议本身变化时，才同步更新 `docs/online-mode/`。
- 首页展示和公告入口涉及全局 UI 时，可以从 `docs/ui-design/` 链接到本文档，但不要重复维护公告类型、维护生命周期和服务端限制范围。
- 生产发布步骤、迁移执行顺序和回滚操作仍以 [生产发布 Runbook](../production-release-runbook.md) 为准；本文档只记录平台状态与公告能力本身的设计现状。
