全局样式最小化	body 只放真正全局的样式（字体、颜色），不放布局约束

布局稳定性依赖于显式的高度传递链，必须为 Flex 子项补充 min-h-0、绝对定位元素添加 h-full 并预留滚动边距，杜绝容器撑破或滚动失效。交互体验的提升需遵循结构化原则，将动画同步转化为 DOM 层级嵌套、响应式切换改为条件渲染，并警惕 transform 造成的层叠上下文隔离。故障排查时应坚持数据驱动，利用 DevTools 直接验证盒模型计算尺寸而非依赖肉眼猜测，精准定位每一层的真实宽高与上下文边界。
