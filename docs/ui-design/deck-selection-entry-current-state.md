# 对局入口卡组选择现状与未落地项

> 文档类型: 专题说明
> 适用范围: 游戏准备页、正式联机房间页、联机调试页的卡组选择与开始/锁组路径
> 最后更新: 2026-07-24
> 当前状态: 描述当前实现事实与明确未落地能力；代码行为以相关代码路径为准

## 当前范围

对局入口选组体验覆盖以下页面：

- 对墙打和本地双人调试准备页，代码路径为 `client/src/components/pages/GameSetupPage.tsx`。
- 正式联机房间准备页，代码路径为 `client/src/components/pages/OnlineRoomPage.tsx`。
- 联机调试准备页，代码路径为 `client/src/components/pages/OnlineDebugPage.tsx`。
- 共用卡组选择控件，代码路径为 `client/src/components/common/DeckSelector.tsx`。

当前目标是降低进入对局前的重复选择和长列表滚动成本，同时保持锁组、准备和开始对局仍由玩家显式操作。

## 当前已实现

### 入口级上次使用卡组

前端会在本地记录部分入口的上次成功使用卡组：

- 正式联机：成功锁组时记录当前云端卡组。
- 对墙打：成功开始对墙打时记录己方卡组。
- 本地调试：成功开始本地调试时分别记录 Player 1 / Player 2 卡组。
- 联机调试：成功锁定调试卡组时按 match 和 seat 记录。

卡组列表加载完成后，页面会按当前入口尝试恢复默认选择。默认选择只更新选中状态，不会自动锁组、自动准备或自动开始。

默认选择的回退顺序为：

1. 上次使用卡组仍存在且合法。
2. 只有一副合法卡组时选中该卡组。
3. 多副合法卡组时选中列表中的最新合法卡组。

### 共用紧凑列表

`DeckSelector` 支持 `comfortable` 和 `compact` 两种密度。对局入口当前使用紧凑模式。

紧凑模式的当前行为：

- 条目高度更低，列表可在有限高度区域内滚动。
- 条目保留卡组名、选中态、上次使用标记、成员/LIVE/能量数量和点数。
- 合法卡组上下文中不重复展示“可用”标签。
- 不合法卡组仍展示“不完整”状态，并禁用选择。
- 选中卡组有边框、顶部强调线和勾选图标。

`DeckSelector` 刷新或默认选中变化时不会重新播放整列入场动画；条目删除或离开时仍保留 exit 动画。

### 正式联机准备页

正式联机页在已进入房间的准备区域中，把当前卡组摘要和主操作放在卡组列表之前。

当前布局行为：

- 窄屏下先显示房间状态、当前卡组摘要、锁组/准备/离开等操作，再显示“更换卡组”列表。
- 桌面下使用左右布局，卡组列表有明确高度，主操作区不会被列表内容挤到视口外。
- 锁组后的当前卡组摘要只在所选显示项确实匹配锁定卡组时展示统计，避免锁组后点击其他卡组导致名称和统计不一致。

### 联机调试准备页

联机调试页与正式联机页使用相同的信息层级：

- 当前 match 和 seat 状态优先展示。
- 当前调试卡组摘要和锁组/重置操作位于卡组列表之前。
- “更换调试卡组”列表使用紧凑模式和有限高度滚动。

### 对墙打与本地调试准备页

`GameSetupPage` 仍保留分步流程：

- 对墙打选择己方卡组后进入确认页。
- 本地双人调试依次选择 Player 1 和 Player 2 卡组后进入确认页。
- 当前步骤里的卡组选择器使用紧凑模式。
- 页面会按入口和玩家位恢复默认卡组。

## 当前未落地

### 服务器端用户偏好

上次使用卡组目前只保存在浏览器本地。不同设备、不同浏览器或清空本地存储后不会同步。

### 折叠式更换卡组入口

当前仍直接展示紧凑卡组列表，没有把完整列表折叠到“更换卡组”按钮或抽屉之后。

### 推荐卡组摘要区

当前默认选中会显示在现有摘要和列表中，但没有单独的“推荐使用”摘要卡。

### 大列表筛选

对局入口的选组列表当前不提供入口内搜索、“只看最近更新”或其他轻量筛选。列表仍依赖 `buildDeckDisplayItems` 的排序结果。

### Sticky 主操作区

对墙打和本地调试仍使用现有分步布局和底部导航，没有为卡组选择步骤新增独立 sticky 主操作区。

正式联机和联机调试已把主操作提前到列表之前，但没有为所有视口提供始终吸附的操作栏。

### 对局入口专项 Playwright 视口验收

当前 `client/tests/e2e/mobile-layout.spec.ts` 已用 mock API 覆盖四种移动端/平板视口的页面级横向溢出与截图基线，但尚未补齐对局入口卡组选择的专项交互断言。

待覆盖的视口场景包括：

- 390 x 844 下正式联机准备页不滚动即可看到当前卡组和锁组按钮。
- 390 x 844 下对墙打选中卡组后不滚动即可看到明确的继续操作。
- 1366 x 768 下正式联机和联机调试卡组列表不会把主操作区挤出视口。
